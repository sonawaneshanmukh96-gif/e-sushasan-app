# Nandurbar e-Sushasan Application

This is the Next.js scaffold for the Nandurbar e-Sushasan Admin System.

## Prerequisites
- Node.js 18+
- npm

## Setup Instructions

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Access the App**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features Implemented
- **Admin Sidebar**: Navigation to all modules.
- **Visual Organogram**: Recursive view of district administration.
- **Department Dashboard**: "Oversight" view with Subsection filtering (Pills).
- **Verification Logic**: Simulation of Task status transitions in `src/lib/logic.ts`.
- **Supporting Documents**: Reusable component at the bottom of pages.
